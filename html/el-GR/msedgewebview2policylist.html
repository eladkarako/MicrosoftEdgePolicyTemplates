Microsoft Edge WebView2 - Πολιτικές

Η τελευταία έκδοση του Microsoft Edge WebView2 περιλαμβάνει τις ακόλουθες πολιτικές. Μπορείτε να χρησιμοποιήσετε αυτές τις πολιτικές για να ρυθμίσετε τον τρόπο με τον οποίο θα εκτελείται το Microsoft Edge WebView2 στον οργανισμό σας. Για πληροφορίες σχετικά με ένα πρόσθετο σύνολο πολιτικών που χρησιμοποιούνται για τον έλεγχο του τρόπου και της χρονικής στιγμής ενημέρωσης του Microsoft Edge WebView2, ανατρέξτε στο θέμα Αναφορά πολιτικής ενημέρωσης του Microsoft Edge.

ΣΗΜΕΙΩΣΗ

Αυτό το άρθρο ισχύει για την έκδοση Microsoft Edge 87 ή νεότερη έκδοση.

Διαθέσιμες πολιτικές

Αυτοί οι πίνακες παραθέτουν όλες τις πολιτικές ομάδας που είναι διαθέσιμες σε αυτήν την έκδοση του Microsoft Edge WebView2. Χρησιμοποιήστε τις συνδέσεις του πίνακα για να δείτε περισσότερες λεπτομέρειες σχετικά με συγκεκριμένες πολιτικές.

Επιστροφή στην αρχή

BrowserExecutableFolder

Ρύθμιση παραμέτρων της θέσης του εκτελέσιμου φακέλου του προγράμματος περιήγησης

Υποστηριζόμενες εκδόσεις:
  • Σε Windows από 87 ή νεότερο
Περιγραφή
Αυτή η πολιτική ρυθμίζει τις παραμέτρους WebView2 εφαρμογών για να χρησιμοποιήσετε το WebView2 Runtime στην καθορισμένη διαδρομή. Ο φάκελος θα πρέπει να περιέχει τα ακόλουθα αρχεία: msedgewebview2.exe, msedge.dll και ούτω καθεξής.

Για να ορίσετε την τιμή για τη διαδρομή του φακέλου, πληκτρολογήστε το όνομα της τιμής και το συνδυασμό τιμής. Ορίστε το όνομα της τιμής στο όνομα του μοντέλου χρήστη της εφαρμογής ή στο όνομα του εκτελέσιμου αρχείου. Μπορείτε να χρησιμοποιήσετε το χαρακτήρα μπαλαντέρ "*" ως όνομα τιμής για να εφαρμοστεί σε όλες τις εφαρμογές.
Υποστηριζόμενες δυνατότητες:
  • Μπορεί να είναι υποχρεωτικό: Ναι
  • Μπορεί να προταθεί: Όχι
  • Ανανέωση δυναμικής πολιτικής: Ναι

Επιστροφή στην αρχή

ExperimentationAndConfigurationServiceControl

Έλεγχος επικοινωνίας με την υπηρεσία πειραματισμού και ρυθμίσεων

Υποστηριζόμενες εκδόσεις:
  • Σε Windows από 97 ή νεότερο
Περιγραφή
H υπηρεσία πειραματισμού και ρυθμίσεων χρησιμοποιείται για την ανάπτυξη ωφέλιμων φορτίων πειραματισμού και ρύθμισης παραμέτρων στον πελάτη.

Το ωφέλιμο φορτίο πειραματισμού αποτελείται από μια λίστα με τις πρώιμες δυνατότητες που αναπτύσσονται από τη Microsoft με σκοπό τη δοκιμή και την παροχή σχολίων.

Το ωφέλιμο φορτίο ρύθμισης παραμέτρων αποτελείται από μια λίστα προτεινόμενων ρυθμίσεων που θέλει να αναπτύξει η Microsoft ώστε να βελτιστοποιήσει την εμπειρία χρήστη.

Το ωφέλιμο φορτίο ρύθμισης παραμέτρων μπορεί επίσης να περιλαμβάνει μια λίστα ενεργειών που μπορούν να εκτελεστούν για λόγους συμβατότητας. Για παράδειγμα, το πρόγραμμα περιήγησης μπορεί να παρακάμπτει τη συμβολοσειρά παράγοντα χρήση σε μια τοποθεσία web, αν η συγκεκριμένη τοποθεσία είναι κατεστραμμένη. Κάθε μία από τις ενέργειες αυτές εφαρμόζονται προσωρινά όσο προσπαθεί η Microsoft να λύσει το πρόβλημα σε συνεργασία με τον κάτοχο της τοποθεσίας.

Αν ορίσετε αυτή την πολιτική σε "FullMode", θα γίνεται λήψη του πλήρους ωφέλιμου φορτίου από την υπηρεσία πειραματισμού και ρυθμίσεων. Σε αυτό περιλαμβάνονται τόσο το ωφέλιμο φορτίο πειραματισμού όσο και το ωφέλιμο φορτίο ρύθμισης παραμέτρων.

Αν ορίσετε αυτή την πολιτική σε "ConfigurationsOnlyMode", θα κατεβαίνει μόνο το ωφέλιμο φορτίο ρύθμισης παραμέτρων.

Αν ορίσετε αυτήν την πολιτική σε "RestrictedMode", θα διακοπεί πλήρως η επικοινωνία με την υπηρεσία πειραματισμού και ρυθμίσεων.Η Microsoft δεν συνιστά αυτήν τη ρύθμιση.

Αν δεν ρυθμίσετε τις παραμέτρους αυτής της πολιτικής, σε μια διαχειριζόμενη συσκευή, η συμπεριφορά στα κανάλια Stable και Beta θα είναι ίδια με τη λειτουργία "ConfigurationsOnlyMode". Στα κανάλια Canary και Dev η συμπεριφορά είναι ίδια με τη λειτουργία "FullMode".

Αν δεν ρυθμίσετε τις παραμέτρους αυτής της πολιτικής, σε μια μη διαχειριζόμενη συσκευή, η συμπεριφορά θα είναι ίδια με τη λειτουργία "FullMode".

Αντιστοίχιση επιλογών πολιτικής:

* FullMode (2) = Ανάκτηση ρυθμίσεων παραμέτρων και πειραματισμού

* ConfigurationsOnlyMode (1) = Ανάκτηση μόνο ρυθμίσεων παραμέτρων

* RestrictedMode (0) = Απενεργοποίηση επικοινωνίας με την υπηρεσία πειραματισμού και ρυθμίσεων

Χρησιμοποιήστε τις προηγούμενες πληροφορίες κατά τη ρύθμιση των παραμέτρων αυτής της πολιτικής.
